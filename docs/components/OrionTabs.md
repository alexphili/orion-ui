# OrionTabs

`<o-tabs>` displays tabs. It must be used with `<o-tab-pane>` components ([more infos below](#orion-tab-pane)).

Its mecanism is very similar with the `<o-timeline>` component ([more infos here](../components/OrionTimeline.md)).
